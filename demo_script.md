# Demo Walkthrough Script (Extended)

## 1. Herb Explorer & Molecular Depth
**Action**:
1. Go to "Herb Explorer".
2. Select an ingested herb (e.g., *Panax ginseng* if in dataset, or *Glycyrrhiza glabra*).
3. Scroll to **Compounds**.
   - Show the table with **SMILES**, **MW**, **LogP**.
   - "We now ingest rigorous chemical properties."
4. Scroll to **Targets & Diseases**.
   - "This herb targets 15 proteins linked to inflammation."

## 2. Prescription Formulas (New)
**Action**:
1. Go to "Formulas".
2. Select a formula (e.g. *Si Jun Zi Tang* or whatever is in `prescription.xlsx`).
3. View **Ingredients**.
   - "We see the exact dosage and role of each herb."
4. View **Mechanistic Expansion**.
   - "The system automatically mapped these 4 herbs to 150 compounds, hitting 20 unique protein targets."
5. Click **Export Rationale**.
   - Download the JSON evidence package.

## 3. Combination Synergy (Extended)
**Action**:
1. Go to "Combination Reasoning".
2. Select *Guduchi* and *Turmeric*.
3. Analyze.
4. Show **Mechanistic Synergy**.
   - "They share 3 protein targets (e.g., COX-2)."
   - "This proves multi-target synergy beyond simple additive effects."

## 4. Disease Network
**Action**:
1. Go to "Diseases".
2. Select a disease (e.g. *Diabetes*).
3. Show **Associated Targets**.
4. Show **Potential Treatments**.
   - "These herbs contain compounds that target the same proteins involved in Diabetes."
